## 2. Email Chat Assistant Agent

### Overview

Generates professional emails from user prompts using Google Gemini.

### Workflow

Manual Trigger → Edit Fields → AI Agent → Google Gemini Chat Model

### Example Prompt

write a professional leave request email for 2 days

### Features

* Professional email generation
* Customizable prompts
* Gemini-powered responses
